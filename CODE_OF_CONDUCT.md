# Code de Conduite

Ce projet adopte le [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Résumé

Respect. Empathie. Pas de harcèlement, pas d'attaques personnelles, pas de
discrimination. Critique sur le code, pas sur la personne.

## Signalement

GitHub en attendant). Réponse confidentielle sous 72h.
